Introduction:

Class Sniper is an ambitious web project created with the purpose to reserve a position in an open class when the class becomes available. This idea was born from the frustration of registration. After classes fill, some classes provide a FIFO (First In First Out) list that students may join. Sadely most classes do not have this approach. This "free-for-all" requires student to sit by their computer day and night refreshng the page until the class is open. In order to evolve from this primitave game of "refresh and seek", Class Sniper was made to automate the task.
